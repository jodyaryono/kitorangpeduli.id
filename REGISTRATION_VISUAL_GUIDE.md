## Responsive Breakpoints

### Desktop (> 768px)
- 2-column grid for inputs
- Max width: 1024px
- Side-by-side form fields

### Tablet (768px - 1024px)
- 2-column grid maintained
- Slightly narrower spacing

### Mobile (< 768px)
- Single column layout
- Full-width inputs
- Stack all form fields
- Touch-friendly buttons

## Animations

1. **Step Transition**: Slide in from right (0.3s)
2. **Input Focus**: Subtle scale up (0.2s)
3. **Button Hover**: Color transition (0.15s)
4. **Progress Update**: Smooth color change (0.3s)
5. **Image Upload**: Fade in preview (0.2s)
